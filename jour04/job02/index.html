Fonction jsonValueKey()

Extraction de valeurs depuis une chaîne JSON

Description :

La fonction jsonValueKey(jsonString, key) prend en paramètre :

  • jsonString : Une chaîne de caractères au format JSON
  • key : La clé dont on veut récupérer la valeur

Elle retourne la valeur associée à cette clé dans la chaîne JSON.

Exemple donné :

JSON :
{
  name: "La Plateforme_",
  address: "8 rue d'hozier",
  city: "Marseille",
  nb_staff: "11",
  creation: "2019"
}

Clé : "city"
Résultat attendu : "Marseille"
